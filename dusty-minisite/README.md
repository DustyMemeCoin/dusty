# DUSTY minisite (GitHub Pages)

This is a tiny one-page landing page for DUSTY. It exists mainly to satisfy explorer/listing forms that require a project website.

## Quick setup
1. Create a new GitHub repo (public): `dusty`
2. Upload all files from this folder to the repo root
3. Repo Settings â†’ Pages:
   - Source: Deploy from a branch
   - Branch: `main` / `(root)`
4. Your site will appear at:
   - `https://YOUR_GITHUB_USERNAME.github.io/dusty/`

## Customize
Open `index.html` and replace these placeholders:
- `REPLACE_WITH_MINT_ADDRESS`
- `REPLACE_WITH_X_LINK` (can be your X profile)
- `REPLACE_WITH_DEXSCREENER_LINK`
- `REPLACE_WITH_SOLSCAN_LINK`
- `REPLACE_WITH_RAYDIUM_LINK`
- `REPLACE_WITH_JUPITER_LINK`
- `REPLACE_WITH_TELEGRAM_LINK`

No build tools needed.
